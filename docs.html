Basic Usage

Page updated: 1/21/2020




Typically, the Codesheet will be formatted in a certain way. We need to adjust the formatting so that data is ordered in a way Quotus can understand.

Generally, Quotus expects data to look like this template:

Quota Group Name
Quota1 Percentage/Or N Question in Survey/Precode Name Code
Quota2 Percentage/Or N Question in Survey/Precode Name Code
Quota3 Percentage/Or N Question in Survey/Precode Name Code

Here is an example with real data:

Gender
male 45% pGender 1
female 51% pGender 2
other 4% pGender 3



Once you've set up your spreadsheet like this, copy the sections you need quotas for:
Paste them into the input space:
Next, fill in the form for your desired output. For projects which aren't trimode or dualmode, those fields can be left blank (along with the Client name).
Once the process button is clicked, you want to scroll down into the Report section and see if all the fields are inputted correctly. You will get a warning for not filling in Mode fields, but that's OK for Phone projects. It serves as a reminder.
Upon reading the warning, you can click the continue button if everything is intended. This will proceed to validate the data and create a CSV sheet.


When the data is processed, buttons will appear in the Report section of the page. One is a toggle-based button which will display the data for viewing on the webpage, the other allows you to export your data as a csv file for direct importing with Limesurvey. Before you do either, it's important you read and understand the warnings provided.






Group Level Customizations


These are modifiers you can add infront of Quota Group names to export them differently.

Name Label Functionality Example
Tri-Split (tri) All Quotas in the group will be treated as Trimode quotas. The Trisplit will be used and quotas will additionally be represented as Phone, Email, and Text. (tri)Party
Dual-Split (dual) All Quotas in the group will be treared as Dualmode quotas. The Dualsplit will be used and quotas will additionally be represented as Phone and Email. (dual)Gender
Raw N Values (raw) All Quotas in the group will treat the percentage field as raw number to use as the Quota limit. (raw)PhoneType
UnSplit (us) When Split Quotas are enabled, this will not generate split quotas for the specific Group. (us)SplitAB
Flex percentage (flex N%) Applies an N% Flex to the Quota Limit. The Quota's name will be modified to indicate it's added Flex. (Flex 6%)Region
Lake Table (lake RegionCount) Include at the start of a formatted table to mark it as a Lake Table.
